# Projects
A multi-phase MySQL relational database system designed for an auto rental point-of-sales workflow, featuring normalized schemas, validation testing, and full transaction documentation.
EZRental is a database-driven point-of-sales management system built in MySQL Workbench. This project focuses on designing and implementing a scalable relational schema for rental transactions, credit card processing, and business operations, complete with normalization, referential integrity, and full schema documentation.
